# 🚀 CloudDeploy Local

> A Self-Hosted Platform-as-a-Service (PaaS) built with Docker, Kubernetes, GitHub Actions, Floci, and a modern observability stack.

## 📌 Overview

CloudDeploy Local is a production-inspired deployment platform that enables developers to build, deploy, monitor, and manage containerized applications entirely on a local machine.

The platform integrates Docker Desktop Kubernetes, GitHub Actions, a local Docker Registry, Prometheus, Grafana, Loki, and Floci to simulate a real-world cloud deployment environment.

---

## ✨ Features

- 🐳 Dockerized Microservices
- ☸️ Kubernetes Deployment Engine
- 🔄 Automated CI/CD with GitHub Actions
- 📦 Local Docker Registry
- 📊 Prometheus + Grafana Monitoring
- 📜 Centralized Logging with Loki
- ☁️ Local AWS Services using Floci
- 🔁 Automatic Rollback
- 📈 Scaling & Resource Monitoring
- 🔐 Kubernetes Secrets & ConfigMaps
